# Todo App Handson

タスク管理アプリケーションです。

## 概要

シンプルなタスク管理アプリで、ブラウザ上でタスクの追加・完了・削除が行えます。
データはブラウザの LocalStorage に保存されるため、サーバー不要で動作します。

## 技術スタック

| 技術 | 用途 |
|------|------|
| HTML | UIの構造定義 |
| CSS | スタイリング・レイアウト |
| JavaScript | インタラクションとロジック |
| LocalStorage | タスクデータの永続化 |

## 動作環境

- モダンブラウザ（Chrome / Firefox / Safari / Edge）
- サーバー不要（ファイルをブラウザで直接開いて動作）

## 使い方

1. `index.html` をブラウザで開く
2. テキストボックスにタスクを入力して追加
3. タスクをクリックして完了/未完了を切り替え
4. 削除ボタンでタスクを削除
